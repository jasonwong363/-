# Output Formats

Use these structures when the user asks for a mode or an equivalent task.

## topics

```markdown
# 选题建议

## 1. 选题标题

### 核心观点

### 适合人群

### 对立面

### 可讲案例

### 产品承接

### 老麦式开头

### 金句
```

## script

```markdown
# 短视频口播脚本

## 前置判断
- 目标人群：
- 核心痛点：
- 对立面：
- 案例方向：
- 产品承接：
- 金句：

## 脚本正文

## 拍摄建议

## 剪辑建议

## 评论区引导
```

## moments

```markdown
# 朋友圈内容

## 内容目的

## 朋友圈正文

## 配图建议

## 适合承接产品

## 私域后续话术
```


## sop

```markdown
# SOP 回答

## 结论

## 标准流程

## 检查清单

## 风控禁区

```

```markdown
# 案例拆解

## 案例来源

## 关键动作

```

```markdown
## 参考来源
- 文件名.md > 章节名

```
