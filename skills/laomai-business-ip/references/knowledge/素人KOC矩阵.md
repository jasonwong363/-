# 素人KOC矩阵

## 模板一：求助帖

用户主动求推荐，求推荐的语气发内容，评论区自然出现商家信息。

**格式：**在XX做了XX，求推荐 / YY哪家好？

**注意：**标题一定要加区域关键词（如"杭州婚纱照"），正文和话题标签都要埋入地域词。

钓鱼帖不带品牌词——带了就等于明牌广告。

## 模板二：反馈型种草帖

已消费用户的真实反馈分享。

**格式：**我在XX家做了XX，效果是XXX（附带使用感受）

**适合行业：**所有实体商家，本地生活的核心模板。

**关键配比：**不带品牌名只拍摄店内风格——防止被系统识别为广告，也防止被同行发现你在做种草。

## 模板三：合集种草帖

ABCD多款产品 / 方案汇总，满足同一类人群的不同需求。

**格式：**备婚清单 / 装修攻略 / XX选购指南

举例：无痛备婚超详细攻略；装修最全攻略——板材名称、环保等级，品牌推荐。

**模板迁移：**把婚纱摄影的合集模板迁移到月子中心、定制珠宝、健身课程等所有高客单行业都可以用。

## 模板四：测评对比帖

用自己的使用体验做真实测评。

**格式：**买之前做了很多功课，最后选了XX，原因如下

**适合行业：**珠宝、培训、健身、美业等所有需要建立信任的品类。

用户评论维护是关键：评论委要主动引导讨论，植入商家信息，不能只有问链接的问链接。

## 五、素人种草的评论区运营

评论区比正文更重要——70%的种草效果来自评论区。

### 评论区布局标准

| 点赞量 | 布局要求 |
| --- | --- |
| 10个赞以下 | 至少1条反馈 + 1条询问（并给链接） |
| 10-50个赞 | 反馈 + 询问 + 官方回复都要有 |
| 50个赞以上 | 多种类型评论配合 |

### 评论类型

- 反馈型：真实用户的购买体验（第一人称，有细节）

- 询问型：问价格 / 问品牌 / 问联系方式

- 情绪型：引发共鸣，引发讨论，提升互动数据

- 引导型：引导用户关注、点赞、评论

### 楼中楼埋词技巧

在评论一楼回复中植入商家信息：用户问"在哪家"，用小号回复"上面那家XXX"，形成自然推荐。

### 官方回复的重要性

官方回复能增加信任感。回复要有差异化，并且不要是营销型的内容，不要用同一条话术在所有帖子下面重复——系统会识别同质化内容。
